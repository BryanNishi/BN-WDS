Thanks to my 20 years in the live entertainment industry, I am a quick learner with quick problem solving and troubleshooting skills. DM's Database React App

This was my final project for the DU bootcamp. Me and a classmate created this Dungeons and Dragons App to help run and play a game. Features include, generators using mongoDB for pulling values, public API for DnD 5e stats and info, javascript dice roller, and MVC file structure. Notable Technologies: React.js, REST API, MongoDB, React-Drag-and-Drop, and React-To-Print. Difficulties on this project were learning React's MVC structure, melding javascript and JSX to function like vanilla JS, dynamically creating and updating stateful components in the character creator, and getting the print function to work.
